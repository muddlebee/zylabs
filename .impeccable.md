# Design Context — Zylabs AI Research Copilot

## Product
AI Research Copilot that helps sales reps prepare for meetings by generating structured company briefings via a LangGraph pipeline.

Tagline: "Your sellers run the conversation. We do everything else."

## Target Audience
B2B sales reps and business development professionals preparing for prospect meetings. Time-pressured — they need scannability and fast signal, not dense prose. But the output (report) needs to feel authoritative and trustworthy.

## Use Cases
1. Enter company name + URL + meeting objective → kick off AI research pipeline
2. Watch the workflow run in real-time (7-node LangGraph pipeline)
3. Read the structured briefing: overview, products, customers, signals, risks, discovery questions, outreach strategy, unknowns
4. Ask follow-up questions via chat grounded in the report

## Brand Personality
- **Intelligence, not flash** — this is a serious tool for serious professionals
- **Editorial** — the report should feel like a real analyst brief, not a JSON dump
- **Efficient** — every second counts; the UI should be fast and never waste space
- Tagline tone: confident, "we handle the research, you handle the relationship"

## Aesthetic Direction
Editorial professional — warm off-white background, deep ink typography, amber/gold accent.
Think: premium internal sales intelligence tool, Bloomberg meets Notion.

- Light mode (intentional — not the lazy AI dark mode)
- Warm off-white background: oklch(97.5% 0.008 75)
- Deep navy text: oklch(15% 0.025 255)
- Amber accent: oklch(72% 0.17 65) — connotes deals, money, premium
- Muted secondaries tinted toward navy
- Instrument Serif for report section headings (editorial weight)
- DM Sans for all UI chrome (legible, not Inter)

## What Must Be Memorable
The live workflow progress — watching nodes light up one by one as the AI pipeline runs.
Make it feel like watching a real intelligence operation unfold.
